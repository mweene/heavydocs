# Javascript Quick Reference

## Arrays
Arrays store ordered list of values

### Common Uses
- Storing multiple values in one variable
- Iterating with loops
- Transforming data

### Summary
Arrays are dynamic, zero indexed, and come with powerful built-in methods

---
## Functions
Functions group related logic.

### Key Points
- Can return values
- Can accept parameters
- Arrow functions offer a shorter syntax

---
## Promises
Promises represent async operations

### Summary
Use '.then()' and '.catch()' or 'async/await' for cleaner syntax
